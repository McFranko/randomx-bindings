# randomx4r

A Rusty wrapper for RandomX hashing.

## Installation

Depends on the [randomx4r-sys](https://github.com/moneromint/randomx4r-sys) crate for FFI.
Building this crate requires the installation of CMake and a C++ compiler.

## Links
